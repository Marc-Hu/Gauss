# Gauss

Ce programme permet de résoudre les systèmes linéaire avec la méthode du pivot de gauss.
Pour cela il faut avoir un fichier .txt avec en en-tête le nombre de colonne (espace) le nombre de ligne (entrer) puis toutes les valeurs de la matrice les uns en dessous des autres. 
Le programme affichera les solutions si toutes les variables sont liés.
Si il y a au moins une variable libre alors le programme affichera la/les variable(s) Xn qui est/sont libre(s).
Taper make dans le terminal pour que le programme se lance automatiquement.
